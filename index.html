About This Project

This project is part of my ICT171 Infrastructure assignment. I built this website using AWS EC2, HTML, CSS, and JavaScript. It is also integrated with WordPress and other cloud tools including Route 53 for DNS and HTTPS for secure access.

The purpose of this website is to highlight the 10 greatest boxers in history, based on skill, record, impact, and legacy. Each profile includes their key achievements and famous quotes. This site is a tribute to boxing legends and serves as a showcase for web development skills including hosting, scripting, and content management systems.
